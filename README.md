# Register → Login → User Profile App

A front-end web application that demonstrates a complete **Register → Login → User Profile** workflow using **HTML, CSS, and JavaScript**, with user data managed through **Browser Local Storage**.

This project focuses on building a clear authentication flow, validating user inputs, and managing protected page access on the client side without using any backend or database.

---

## Features
- User registration with required field validation
- Login using Email or Phone number
- Client-side authentication using Local Storage
- Duplicate user prevention
- User profile / dashboard view
- Protected dashboard access (login state check)
- Logout functionality with session cleanup
- Meaningful error messages for better UX
- Simple and clean user interface

---

## Tech Stack
**Frontend:** HTML, CSS, JavaScript  
**Storage:** Browser Local Storage  
**Deployment:** Vercel

---

## How to Run
1. Download or clone the repository
2. Open `index.html` in a browser
3. Register a new user
4. Login using registered credentials
5. View user details on the dashboard

No server or installation required.
